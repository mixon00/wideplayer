WidePlayer for X

About & Help

WidePlayer makes supported in-feed videos on X wider without fullscreen. Use the popup for quick settings.

About

WidePlayer works on supported in-feed videos on x.com. It moves the original player into a wider overlay while keeping the feed layout stable.

The popup is the main place for quick settings. You can switch between Auto and Manual mode there and adjust the preferred width at any time.

FAQ

Where does WidePlayer work?

On supported in-feed videos on x.com.

Why doesn’t it appear on every video?

Some X layouts are still unsupported. WidePlayer skips videos it cannot enhance safely instead of forcing the UI into a broken state.

What’s the difference between Auto and Manual mode?

Auto mode expands supported videos automatically. Manual mode leaves videos alone until you use the per-video control on the player.

Do width changes apply immediately?

Yes. While you drag the width slider in the popup, active players preview the change live before the final value is saved.

Does WidePlayer collect any data?

No. WidePlayer only stores your local extension settings, such as mode and preferred width.

What’s new

Recent improvements in the current MVP.

  • The popup is now the main place for quick settings.
  • Width changes preview live on active players while you drag the slider.
  • Manual mode supports per-video controls, outside-click collapse, and centered opening scroll.

If WidePlayer does not appear right away after installing or updating, reload X and try again.
