WidePlayer

About & Help

WidePlayer makes supported in-feed videos wider without going fullscreen. Use the popup for quick settings.

About

WidePlayer expands supported in-feed videos into a wider overlay while keeping the feed layout stable. X.com and Mastodon are supported now. Bluesky and LinkedIn are planned.

The popup is the main place for quick settings. You can switch Auto mode for all platforms at once, override X and Mastodon separately, and adjust the preferred width at any time.

FAQ

Where does WidePlayer work?

WidePlayer works on supported in-feed videos. X.com and Mastodon are supported now. Bluesky and LinkedIn are planned.

Which platforms are supported right now?

X.com and Mastodon are supported today. Bluesky and LinkedIn are on the roadmap, but work has not started yet.

Why doesn’t it appear on every video?

Some layouts and embedded players are still unsupported. WidePlayer skips videos it cannot enhance safely instead of forcing a broken layout.

What’s the difference between Auto and Manual mode?

Auto mode expands supported videos as you scroll. Manual mode leaves videos at their normal size until you use the player control. X and Mastodon have separate Auto mode toggles.

Can I control how wide videos get?

Yes. The popup includes a width slider. Active players preview the change while you drag, then save the final value when you release it.

Does WidePlayer collect any data?

No. The browser extension does not collect, transmit, or sell any user data. It only stores your local extension settings, such as mode and preferred width, on your device.

Is WidePlayer open source?

Yes. The project is open source and available on GitHub. The extension is built in public so the code can be reviewed.

What’s new

Recent improvements in the current MVP.

  • The popup is now the main place for quick settings.
  • Width changes preview live on active players while you drag the slider.
  • Manual mode supports per-video controls, outside-click collapse, and centered opening scroll.

If WidePlayer does not appear right away after installing or updating, reload the supported platform and try again.
